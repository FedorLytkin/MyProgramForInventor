Комментарий к занятию 2

В этом руководстве мы рекомендуем использовать Microsoft Visual Basic 2010 Express.  
Файлы проекта для занятия 2 содержатся в папке lesson2_VisualExpress2010. 
Обратите внимание на 4 файла:

• MyFirstInventorPlugin_Lesson2.vbproj - 
  Файл проекта. Его можно открыть в Microsoft Visual Basic 2010 Express.
	
• Form1.vb 
  Содержит код формы, который должен быть создан на занятии 1.

• Form1.Designer.vb 
  Служебный файл формы.

• Папка "My Project"
  Содержит файл AssemblyInfo, он описывает файлы и свойства проекта, которые используются формой.


Примечание:
Данный проект Microsoft Visual Studio используют ссылку на сборку взаимодействия Autodesk.Inventor.Interop.DLL. Версия этой DLL должна соответствовать используемой вами версии Autodesk Inventor. Вы можете изменить ссылку на этот компонент в настройках проекта Visual Studio.
